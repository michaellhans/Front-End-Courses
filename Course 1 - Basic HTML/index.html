Mata Kuliah di Semester 5 Teknik Informatika ITB

IF3110 - Pemrograman Aplikasi Berbasis Web

Perkuliahan ini mengajarkan prinsip-prinsip dasar Intrenet dan Aplikasi Internet. Kuliah ini mengajarkan sejumlah teknologi yang dapat digunakan untuk membangun sebuah Aplikasi Internet.

IF3130 - Jaringan Komputer

Pengantar ke jaringan computer, perangkat keras jaringan computer & data link layer, model dasar dari jaringan dgn switch, protocol kerja internet, layer transport, kendali congestion dan alokasi sumber daya, layer aplikasi dan keamanan jaringan

IF3140 - Manajemen Basis Data

Mata kuliah ini memberikan pengetahuan mengenai pengelolaan basis data, mencakup peningkatan performansi basis data, manajemen kemanan dan transaksi, pemrograman basis data, dan pengelolaan terdistribusi.

IF3141 - Sistem Informasi

Pada kuliah ini diberikan pengertian, pemahaman dan pengetahuan mengenai konsep sistem informasi dan peranannya dalam meningkatkan keunggulan kompetitif organisasi

IF3150 - Manajemen Proyek Perangkat Lunak

Kuliah ini memberikan pengetahuan dan pemahaman mengenai manajemen, manajemen proyek, proses pada manajemen proyek perangkat lunak , serta beberapa area keilmuan di bidang manajemen proyek perangkat lunak

IF3151 - Interaksi Manusia Komputer

Kuliah ini membahas mengenai konsep desain, pengenalan karakteristik pengguna, aspek praktis dan penelitian terkait antarmuka pengguna, pengumpulan data, framework analisis, proses desain interaksi, dan berbagai pendekatan untuk mengevaluasi antarmuka interaktif.

IF3170 - Inteligensi Buatan

Kuliah ini membahas intelligent agent, mulai dari agen dengan tingkat yang paling sederhana (hanya melakukan pencarian), hingga agen yang mampu membentuk pengetahuan sendiri. Untuk setiap tingkatan diberikan aplikasi yang memanfaatkan agen pada tingkat tersebut. Dalam kuliah ini juga diberikan pengantar untuk pattern recognition.
